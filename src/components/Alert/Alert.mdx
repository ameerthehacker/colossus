# Alert

## Basic Usage

You can use the **variant** prop to change the look of the button

<Alert>This is a primary alert</Alert><br/>
<Alert variant="secondary">This is a secondary alert</Alert><br/>
<Alert variant="success">This is a success alert</Alert><br/>
<Alert variant="warning">This is a warning alert</Alert><br/>
<Alert variant="danger">This is a danger alert</Alert><br/>

## Dismissible Alert

You can use the **dismissible** prop to make an alert dismissble

<Alert dismissible={true} variant="warning">You can dismiss this warning safely!</Alert><br/>
